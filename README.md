## Teste Tripletech

Desafio teste para empresa Tripletech.

#### Util

```

Entre na pasta back-end/server e renomeie o arquivo ".env.example" para ".env"

```

#### Iniciar Back-End

```
-> ~/cd back-end/server

-> ~/yarn ou npm install

-> ~/yarn dev
```

#### Iniciar Front-End

```
-> ~/cd front-end/app

-> ~/yarn ou npm install

-> ~/yarn start
```

#### Desfrute da aplicação! :)
